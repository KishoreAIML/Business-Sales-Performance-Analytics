# 📊 Business Sales Performance Analytics Dashboard

## 📌 Overview

The **Business Sales Performance Analytics Dashboard** is an interactive data visualization project designed to analyze and monitor key business metrics such as sales, profit, and revenue. It provides actionable insights through dynamic charts and KPIs to support data-driven decision-making.

---

## 🚀 Features

* 📈 **KPI Metrics**

  * Total Sales
  * Total Profit
  * Total Revenue
  * Month-over-Month Growth %

* 🌍 **Regional Analysis**

  * Profit distribution across regions (West, East, Central, South)

* 🧩 **Category Insights**

  * Profit split by categories:

    * Office Supplies
    * Furniture
    * Technology

* 📊 **Sub-Category Revenue**

  * Detailed revenue breakdown for sub-categories like:

    * Phones, Chairs, Accessories, etc.

* 🎛️ **Interactive Filters**

  * Ship Mode
  * Year
  * Segment
